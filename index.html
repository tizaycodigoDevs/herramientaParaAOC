Herramienta para AOC - Recent Stories

Esta herramienta genera automáticamente el HTML de Recent Stories a partir de los contenidos del blog.

Pasos rápidos
  1. Presioná Generar HTML.
  2. (Opcional) Previsualizá el resultado.
  3. Copiá o descargá el código generado.
  4. Pegalo en la sección Recent Post del blog.
¿Tenés dudas? Ver instrucciones detalladas
  1. Ingresá a este sitio.
  2. Presioná Generar HTML y esperá unos segundos.
  3. El código HTML se generará en el campo inferior.
  4. Si querés, presioná Previsualizar para ver cómo se verá publicado. Esta acción no modifica el blog.
  5. Presioná Copiar o Descargar .txt. (Descargar sirve como respaldo.El nombre de archivo corresponde a la fecha actual)
  6. Ingresá al administrador del blog.
  7. Entrá en Pages → Recent Post.
  8. Ubicate arriba del bloque de contenido.
  9. Presioná Edit → Code.
  10. Seleccioná todo el contenido del bloque.
  11. Pegá el código generado desde esta herramienta, reemplazando completamente el contenido anterior.
  12. Guardá los cambios.

⚠️ Importante: siempre reemplazá todo el contenido del bloque. No pegues el código encima de otro texto.

El archivo descargado se guarda con el nombre de la fecha actual. Lo podés encontrar en tu carpeta de "descargas o download"
